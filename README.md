# Jogo da Palavra Secreta

Um jogo de adivinhação de palavras desenvolvido com React, onde o jogador precisa descobrir a palavra secreta letra por letra antes de esgotar suas tentativas.

## Sobre o Projeto

O Jogo da Palavra Secreta é uma aplicação web que desafia o jogador a adivinhar palavras de diferentes categorias. A cada rodada, uma palavra aleatória é escolhida e o jogador deve tentar descobri-la digitando letras. O jogo mantém um sistema de pontuação e permite múltiplas rodadas consecutivas.

## Funcionalidades

- Sistema de categorias de palavras
- Seleção aleatória de palavras
- Sistema de pontuação progressiva (+10 pontos por palavra)
- Sistema de vidas/tentativas (5 tentativas por jogo)
- Reinício automático após vitória
- Tela de game over com pontuação final

## Como Jogar

1. Clique em "Começar o jogo" na tela inicial
2. Uma categoria será exibida como dica
3. Digite uma letra por vez para tentar adivinhar a palavra
4. Letras corretas aparecerão na palavra
5. Letras erradas reduzirão suas tentativas
6. Acerte todas as letras para ganhar pontos e avançar para a próxima palavra
7. O jogo termina quando suas tentativas chegarem a zero

## Tecnologias Utilizadas

- React
- JavaScript ES6+
- CSS3
- React Hooks (useState, useEffect, useCallback)

## Instalação e Execução

1. Clone o repositório:
```bash
git clone https://github.com/AgaraCarol/SECRET_WORD.git
```

2. Entre na pasta do projeto:
```bash
cd SECRET_WORD/secretword
```

3. Instale as dependências:
```bash
npm install
```

4. Execute o projeto:
```bash
npm start
```

5. Abra o navegador em http://localhost:3000
